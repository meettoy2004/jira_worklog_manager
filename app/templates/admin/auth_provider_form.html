Configuration Help

LDAP

Generic LDAP directory (OpenLDAP, etc.)

  • Server URI: ldap:// or ldaps://
  • Bind DN Template required
  • Base DN for user searches

Active Directory

Microsoft Active Directory

  • Requires domain name
  • Supports UPN and domain\user
  • NTLM authentication

FreeIPA

Red Hat Identity Management

  • Auto-generates bind DN
  • cn=users container
  • Supports ipaUniqueID

Keycloak/OIDC

OAuth2/OpenID Connect

  • Requires client ID and secret
  • Enable Direct Access Grants in client
  • Token-based authentication

For detailed configuration instructions, see SSO_AUTHENTICATION.md
